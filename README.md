# Nest-React-Admin

## 项目介绍

> **Nest-React-Admin**是基于[Nest](https://nestjs.com) 和 [React](https://reactjs.org) 开发的全栈前后端分离的基础平台。

## 技术选型

> - 前端：基于 [React](https://reactjs.org) 和 [Ant Design](https://ant.design/index-cn) 构建基础页面。
> - 后端：基于 [Nest](https://nestjs.com) 快速构建 **RESTFul** 风格的API，[Nest](https://nestjs.com) 是一个基于 `NodeJS` 和 `TypeScript` 的后端框架。
> - 数据库：使用 [MySQL](https://www.mysql.com) 数据库和 [TypeORM](https://typeorm.io) 完成数据的基本操作。
> - 缓存：使用 [Redis](https://redis.io/) 做数据缓存。
> - API文档：使用 [Swagger](https://redis.io/) 自动生成接口文档。
